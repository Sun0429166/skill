# 公关文案与新闻发布 Skill
## 1. 触发场景
当用户存在以下需求时调用本Skill：
1. 企业危机场景：食品安全、平台内容违规、门店管理疏漏，撰写公开道歉声明
2. 公众人物场景：艺人立场争议、明星个人违法失范，产出微博回应
3. 政务场景：安全生产事故、重大自然灾害，发布阶段性官方通报、发布会口径

## 2. 理论框架
### 2.1 亚里士多德修辞三要素
1. Ethos（可信度）：组织/个人身份、责任承担、整改行动、官方资质，构建公众信任
2. Pathos（情感共情）：承认公众负面情绪、换位思考、价值共鸣，消解对立情绪
3. Logos（事实逻辑）：客观可查证事实、时间线、分条整改方案，避免空泛辩解

### 2.2 肯尼斯·伯克新修辞认同理论
公关声明核心目标是建立组织与公众价值认同，三类认同策略：
1. 同情认同：共情公众损失、担忧与不满（危机场景首选）
2. 对立认同：树立共同负面对立面，拉近组织与公众立场
3. 象征认同：输出安全、守法、爱国、生命至上等公共价值观

## 3. 完整工作流程
1. 信息收集阶段：确认主体（企业/明星/政府）、事件经过、受众、核心争议、法律红线
2. 策略判断：匹配修辞三要素+认同策略，确定沟通基调（诚恳/严谨/安抚）
3. 正文生成：选用对应模板输出初稿
4. 风险复核：检查情绪缺失、事实模糊、推卸责任、法律敏感表述
5. 优化输出：调整话术，补充共情段落、可落地整改措施

## 4. 输出模板库
### 模板1：企业危机道歉声明
# XX品牌关于XX问题的致歉声明
各位消费者、媒体朋友：
（1）事实陈述：客观还原事件，不回避问题（Logos）
（2）共情致歉：承认公众担忧与损失（Pathos）
（3）责任承担：主动担责，不推诿甩锅（Ethos）
（4）整改措施：分点列明可落地、可监督的长期优化方案
（5）沟通渠道：公示监督渠道，持续同步进展
落款：企业全称 发布日期

### 模板2：明星/工作室官方回应
正式盖章声明（立场类事件）/微博短文（个人过失类事件）+媒体尖锐提问Q&A

### 模板3：政府事故新闻发布会口径
事实未完全查清前提下，优先安抚群众、公布伤亡与救援进展，区分已核实信息与待调查内容，稳定社会舆论

## 5. 质量评价标准
### 合格标准
1. Ethos达标：主动承担责任，给出具体整改动作，具备官方/当事人身份背书
2. Pathos达标：开篇共情公众情绪，不冷漠、不回避大众伤痛与不满
3. Logos达标：时间线清晰，整改措施有条理、逻辑闭环
4. 认同策略：至少使用同情认同或对立认同，拉近与公众距离
### 不合格典型问题
1. 通篇解释客观原因，缺少道歉、无视公众情绪（Pathos失效）
2. 回避核心事实，整改措施空洞无执行细则，无完成时限
3. 措辞避重就轻，弱化自身错误；缺少长效监督与改进方案
4. 政务通报对监管疏漏、前期失误轻描淡写，群众核心关切回应不足

## 6. 测试提示词
### 测试提示词1
帮我写一份餐饮连锁品牌门店卫生虫害问题的道歉声明，要求真诚但规避法律风险。
> 预期输出：参考杨国福致歉信结构，完整企业致歉声明，包含事发快速赔付、门店停业消杀、全国门店排查、24小时投诉专线、消费者监督机制，规避高额自认赔偿等法律敏感表述。

### 测试提示词2
艺人相关负面舆论全网发酵，需要发布官方公开回应，生成完整盖章声明/微博致歉文案，附带10条媒体尖锐提问配套标准回应Q&A清单。
> 预期输出：艺人官方回应文案分两类，按需适配场景：1.场景A（立场冲突类：品牌辱华、价值观分歧）：正式盖章长声明，清晰表明立场，坚守价值底线，果断终止相关合作，唤起大众情感认同，逻辑清晰罗列事件完整时间线，强化公众人物正向表率作用。场景B（个人言行过失类：发言失当、个人疏忽犯错）：温和真诚微博短文，主动承认自身问题，诚恳向受影响群体致歉，共情大众情绪，不找客观借口、不弱化自身错误。2.配套10组通用媒体尖锐问答：覆盖「事件发生真实原因」「是否存在逃避责任」「后续如何约束自身言行」「怎样给粉丝树立正向示范」「是否会开展自我反思与整改」等通用高频敏感问题；所有回答严格遵循修辞框架：Ethos主动担责、Pathos共情大众情绪、Logos清晰客观陈述事实，全程规避甩锅、洗白、敷衍话术，适配迪丽热巴解约、黄晓明过失致歉等全部艺人公关案例写作逻辑，可复用在任意艺人舆论危机场景。

### 测试提示词3
为应急管理部门撰写工厂爆炸安全生产事故新闻发布会口径，事故调查尚未完成，需要优先安抚遇难家属、回应群众安全恐慌。
> 预期输出：参考浏阳烟花厂爆炸发布会口径，开篇哀悼致歉，公布医疗救治、善后安置措施，同步说明企业管控、全域停产排查等临时安全举措，区分已确认事实与待核查内容。

## 7. 案例调研库
### （一）企业类案例（2个）
#### 案例1：杨国福苏州门店老鼠卫生危机致歉声明
1. 修辞拆解
- Ethos：主动承认加盟监管疏漏不甩锅，公示官方投诉专线，邀请消费者、媒体突击检查，配合市监局监管，官方集团盖章背书。
- Pathos：以“食品安全是企业生命线”共情大众饮食安全焦虑，公开反省传递愧疚。
- Logos：清晰时间线：2小时致歉赔付→门店停业消杀→全国门店彻查→月度常态化监管；四点整改措施形成闭环。
- 认同策略：同情认同，站在消费者健康安全立场共情。
2. 存在缺陷：未公示赔付标准、未限定全国门店排查完成时限，落地逻辑不足。
3. 适用场景：餐饮食品安全类危机公关通稿撰写参考。

#### 案例2：快手网信违规整改致歉声明
1. 修辞拆解
- Ethos：全盘接受监管处罚，主动认领平台技术与管理缺陷，承诺长期接受社会监督。
- Pathos：以“教训极其惨痛”反思不良内容危害，唤起网民对干净网络环境的共同诉求。
- Logos：逻辑链条：技术缺陷→低俗内容泛滥→监管处罚→全面基建排查+长效治理机制。
- 认同策略：同情认同（大众期待清朗网络）+对立认同（共同抵制低俗不良内容）。
2. 存在缺陷：无具体整改周期、审核升级细则，缺少受害用户安抚，情感薄弱。
3. 适用场景：互联网平台内容违规、监管处罚类官方声明撰写参考。

### （二）明星/公众人物类案例（2个）
#### 案例1：迪丽热巴工作室DG辱华品牌解约声明
1. 修辞拆解
- Ethos：以中国企业、中国公民双重身份确立立场，正式发函终止合作，盖章官方声明具备严肃性。
- Pathos：“无比愤慨”调动全民爱国情绪，“祖国不容侵犯”升华民族共同情感。
- Logos：事实逻辑：设计师辱华言论曝光→发函解约→公开发布立场声明。
- 认同策略：对立认同（共同抵制辱华品牌）+同情认同（全体国民维护国家荣誉）。
2. 存在缺陷：缺少对粉丝、国内消费者安抚，无长期价值导向表态。
3. 适用场景：艺人遭遇品牌价值观冲突、辱华争议解约声明参考。

#### 案例2：黄晓明未成年人骑行违法微博致歉
1. 修辞拆解
- Ethos：当事人直面错误，主动承诺前往交警部门接受处罚，以父亲身份普法，承担公众人物示范责任。
- Pathos：表达愧疚，面向全体家长共情儿童出行安全诉求，虚心接受网友监督。
- Logos：完整事实链条：违规骑行被网友指出→公开道歉→主动接受处罚→科普交通法规；明确引用道路安全法条支撑错误定性。
- 认同策略：同情认同，依托家长群体重视孩子安全的共同心理拉近受众距离。
2. 存在缺陷：将违法行为弱化表述为“疏忽”，无家人后续出行约束长效措施。
3. 适用场景：明星个人违法、失范行为公开道歉文案参考。

### （三）政府发布会口径案例（2个）
#### 案例1：浏阳烟花厂爆炸事故新闻发布会
1. 修辞拆解
- Ethos：政府官方权威发布，主动哀悼遇难者、承担属地监管责任，公开伤亡、救援、企业追责完整信息。
- Pathos：表达沉痛自责，公布伤员“一患一策”救治方案，全域停产缓解民众安全恐慌。
- Logos：闭环逻辑：应急救援→医疗善后→企业控制→全域停产+行业安全排查长效整改。
- 认同策略：同情认同（共情伤亡家属悲痛）+对立认同（共同抵制企业安全违规）。
2. 存在缺陷：监管漏洞细节披露不足，停产复工验收标准模糊。
3. 适用场景：工厂安全生产事故、突发伤亡事件官方发布会口径参考。

#### 案例2：郑州7·20特大暴雨灾害国新办阶段性通报
1. 修辞拆解
- Ethos：国新办高级别官方背书，完整披露降雨极值、受灾、救援数据，不回避城市内涝、地铁险情等负面问题。
- Pathos：开篇共情受灾群众苦难，重点说明群众转移、物资抢修、安置保障工作，缓解民众无助恐慌。
- Logos：标准政务逻辑：灾情数据→救援行动→民生保障→长期防汛整改举措。
- 认同策略：同情认同，依托全民共同受灾经历建立情感共鸣。
2. 存在缺陷：前期预警、应急响应疏漏反思篇幅较少，灾后重建、群众理赔方案单薄。
3. 适用场景：重大自然灾害、城市灾害阶段性官方通报撰写参考。
