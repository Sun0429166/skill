# 公关声明、新闻通稿、发布会口径发布前完整自查清单
适用主体：企业官方声明、艺人工作室回应、政府新闻发布会通稿
依托理论：亚里士多德修辞三要素（Ethos/Pathos/Logos）、伯克认同理论
使用场景：危机致歉、价值观立场声明、突发事件官方通报、媒体问答Q&A

## 一、Ethos 可信度全维度核查（40分核心项）
### 1. 主体身份与官方资质检查
1. 是否明确完整发布主体全称（企业集团/工作室/政府部门），无简称模糊表述；
2. 正式声明是否标注盖章、发布日期、官方联系方式（投诉热线、官网、政务渠道）；
3. 发布主体是否主动站出来发声，无推诿给加盟商、第三方、员工、合作方等甩锅行为；
4. 有无明确身份背书：企业法人、艺人本人、属地主管部门等权威身份视角。

### 2. 责任承担核查
1. 是否主动承认自身管理、监管、行为、制度层面存在漏洞，不回避核心过错；
2. 区分主次责任，不把全部问题归因外部客观因素（市场、网友、环境、第三方）；
3. 针对已发生损失，是否主动提出赔付、整改、处罚、追责等承担责任的具体动作；
4. 政府口径是否主动承担属地监管、应急处置、前期预警相关责任，不回避监管短板。

### 3. 第三方监督与合规背书
1. 是否提及配合市场监管、公安、应急、网信等主管部门调查；
2. 是否开放公众监督渠道：投诉专线、媒体突击检查、消费者代表监督、线上反馈入口；
3. 是否承诺后续公开整改进度，定期对外同步进展，不暗箱整改；
4. 法律法规引用准确（交通法规、食品安全法、安全生产条例等），法条无错误。

### 4. 扣分风险点（出现任意一条Ethos大幅减分）
- 通篇归咎外部环境、消费者、网友、合作方，自身无任何反思；
- 主体名称模糊，无官方联系方式、无发布时间；
- 拒绝配合监管部门调查，拒绝公众监督；
- 刻意弱化自身过错，使用“客观因素导致”“个别员工行为与总部无关”等甩锅话术。

## 二、Pathos 情感共情全维度核查（30分核心项）
### 1. 开篇情绪态度核查
1. 危机类文案首段是否包含致歉、愧疚、痛心、惋惜等共情表述；
2. 立场冲突类文案是否贴合大众主流情绪（爱国、安全、公平），传递统一情感；
3. 灾害、安全事故政务文稿是否优先哀悼遇难者、安抚家属，不先讲整改措施。

### 2. 受众共情适配检查（分三类主体）
1. 企业文案：是否站在消费者健康、财产损失角度共情，理解大众担忧；
2. 艺人回应：
   - 立场类：共情全体国民共同价值诉求；
   - 个人过失类：共情粉丝、被影响群众、家长群体；
3. 政府通报：共情受灾群众、伤者家属、周边居民的恐慌、悲痛情绪。

### 3. 沟通姿态核查（倾听与求同）
1. 是否认可网友监督、群众质疑的合理性，无指责网友、拉黑舆论、对抗公众表述；
2. 是否使用“理解大家的担忧”“感谢各位监督指正”等柔和沟通话术；
3. 沟通逻辑是否先认同大众合理诉求，再阐述自身整改方案，不直接对立反驳。

### 4. 七类情感规避（Pathos失效典型问题）
1. 冷漠生硬：全篇只有整改措施，无任何道歉、愧疚、安抚语句；
2. 居高临下：说教式口吻，教育公众、指责受害者；
3. 避重就轻：弱化伤害、淡化事故严重程度；
4. 情绪对立：反驳网友质疑、指责群众小题大做；
5. 缺少人文关怀：事故通报只讲救援数据，不提伤者、家属安抚；
6. 无统一价值共鸣：脱离大众共同关切（食品安全、生命安全、家国尊严）；
7. 过度煽情空洞：只有情绪话术，无实际行动支撑共情。

## 三、Logos 事实逻辑闭环核查（30分核心项）
### 1. 客观事实核查
1. 事件时间线完整清晰：事发、曝光、处置、调查、整改全流程无时间断层；
2. 所有数据、事件细节客观真实，不模糊、不编造、不隐瞒关键信息；
3. 区分「已核实事实」与「仍在调查内容」，不把未证实猜测当作定论；
4. 无模糊化词语：“相关人员”“部分门店”“后续处理”，尽量明确范围、对象。

### 2. 整改措施逻辑核查
1. 整改措施分点罗列，一条问题对应一条解决方案，因果自洽；
2. 每条整改具备三大要素：执行主体、完成内容、时间期限；
3. 形成闭环：短期紧急处置→中期全面排查→长期制度完善，三层逻辑完整；
4. 配套监督机制：如何检验整改效果、如何持续管控，避免一次性整改。

### 3. 文本结构逻辑规范
1. 企业致歉：致歉共情→事实陈述→责任认领→分层整改→监督渠道；
2. 艺人声明：立场/认错→事实梳理→反思→长效约束；
3. 政府事故通报：哀悼安抚→灾情/事故事实→救援救治→善后安置→全域排查整改。

### 4. Logos扣分缺陷清单
1. 整改无时间期限，无执行标准，无验收方式；
2. 事实时间线混乱，颠倒事件发生顺序；
3. 措施空洞，只说“加强管理”“全面整改”，无落地细则；
4. 隐瞒关键信息，回避群众最关心的赔付、追责、复工标准；
5. 逻辑前后矛盾，前文承认问题，后文又解释规避责任。

## 四、伯克认同策略专项检查（加分项）
1. 是否使用**同情认同**：贴合大众共同情感、共同诉求（食品安全、生命安全、爱国、儿童安全）；
2. 是否需要**对立认同**：存在统一对立面（违规企业、辱华品牌、不良网络内容）时是否合理运用；
3. 是否搭建**象征认同（曲同）**：传递安全、守法、爱国、生命至上等公共正向价值观；
4. 单一文案至少使用1种认同策略，优质文案可叠加两种提升传播效果。

## 五、法律风险专项复核（必查，规避公关事故）
### 1. 企业文案法律红线
1. 不随意自认重大过错、重大人身财产损害，避免索赔风险；
2. 赔付内容不写无上限全额赔偿，明确赔付范围、标准、适用人群；
3. 不使用绝对化承诺：“永久杜绝”“绝对安全”“零风险”；
4. 不擅自定性违法，仅客观配合监管部门调查，由官方出具结论。

### 2. 艺人声明法律红线
1. 解约声明不随意诋毁合作品牌，仅表明立场差异；
2. 个人过失不夸大、不编造情节，如实描述行为；
3. 不发表极端、煽动性言论，规避舆论对立激化。

### 3. 政府口径法律红线
1. 未调查完毕不随意定性企业、个人责任；
2. 伤亡、损失数据以官方核实为准，不预估、不猜测；
3. 不提前披露未定论调查结果，避免误导舆论。

## 六、媒体问答Q&A配套自查（如有问答模块额外检查）
1. 所有尖锐问题均有对应稳妥回应，无回避、无可供断章取义的漏洞话术；
2. 回答统一遵循“共情→认责→事实→整改”固定逻辑；
3. 不出现容易引发二次舆情的敏感表述；
4. 针对高频质疑（特权、洗白、敷衍、长效管控）有专门应对话术。

## 七、最终综合评级标准
1. 优秀（80–100分）：Ethos/Pathos/Logos全部达标，两种及以上认同策略，无法律风险，逻辑完整、共情充足、责任清晰；
2. 合格（60–79分）：三大修辞要素基本齐全，存在少量逻辑/情感短板，无重大法律风险；
3. 不合格（60分以下）：甩锅无担当、无视公众情绪、整改空洞、存在法律敏感话术，需全文重写。
