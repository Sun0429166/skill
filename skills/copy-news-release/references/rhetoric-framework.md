# 公关修辞完整理论框架
## 一、亚里士多德修辞三要素
### 1. Ethos（说话者可信度）
核心定义：ethos 指向说话者的可信度，包含身份、资质、责任、善意；在阐述观点前，优先强调说话主体的合法性与资质，塑造说服者道德可靠、良善诚实的形象。
细分维度：
1. 身份资质：官方主体、企业集团、政府部门、艺人工作室、当事人本人等身份背书，盖章、官方联系方式、发布日期均可强化可信度；
2. 责任担当：不推诿、不甩锅，主动承认自身管理、行为、监管层面存在漏洞；
3. 第三方佐证：主动接受监管部门、媒体、消费者代表监督，配合官方调查；
4. 释放善意：尊重受众、关切受害者、流露悲悯，传递愿意解决问题的正向态度。

### 2. Pathos（阅听人情感共情）
核心定义：pathos 指向受众的情感处境，核心手段包含倾听、移情、求同。
1. 七组基础情感维度：恼怒-温和，友善-敌意，恐惧-无畏，羞耻-无耻，感激-忘恩，嫉妒-倾慕，怜悯-憎恨；
2. 移情：依靠双方共同情感、共同意见、统一价值观产生同理心，站在受众视角感受担忧、痛苦、气愤；
3. 倾听策略：沟通中给予公众发声空间，主动接纳网友监督、群众提问，不阻断舆论表达；
4. 赞美技巧：真诚正向表达共识，低成本拉近心理距离；
5. 求同沟通逻辑：对话中先认同大众合理诉求，多用“是”建立共识，再阐述自身方案与立场，减少对立。

### 3. Logos（内容事实逻辑）
核心定义：logos 依托事实、客观证据、完整逻辑推理支撑观点，避免空洞主观话术。
1. 推理方式：归纳、演绎，论证过程自洽；
2. 缺省三段论：借助大众共识作为论证过渡，无需重复基础常识；
3. 二分对立统一套路：区分问题/对策、过错/整改、已核实信息/待调查信息；
4. 落地逻辑要求：清晰时间线、分条整改措施、明确执行周期、量化处置方案，形成闭环流程。

## 二、肯尼斯·伯克新修辞认同理论
核心观点：公关修辞的核心不是单向说服，而是依靠“认同”消解组织与公众之间的隔阂；公关声明不等于单方面辩解，核心是寻找双方共通的价值底线。
三大认同策略：
1. 同情认同
依靠双方一致的情感、意见、思想、价值观达成共情；覆盖情绪、态度、观念、身份多重层面，危机公关首选策略。
2. 对立认同
双方存在矛盾冲突，但拥有同一个共同“对立面”，由此站在同一立场达成统一认知。
3. 象征认同（曲同）
大众对符号、价值观、正向精神寄托产生心理依附；组织通过坚守公共象征价值（爱国、生命至上、食品安全、守法）获取受众长期认同。
