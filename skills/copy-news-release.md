# 公关文案与新闻发布 Skill
## 1. 触发场景
当用户存在以下需求时调用本Skill：
1. 企业危机场景：食品安全、产品质量、数据泄露、员工丑闻，撰写公开道歉声明
2. 公众人物场景：主播/明星直播失言、代言翻车、粉丝争议，产出微博回应+媒体问答
3. 政务场景：安全事故、公共卫生事件，发布阶段性官方通报、发布会口径
4. 常规品牌传播：新品上市、公益活动、品牌合作新闻通稿

## 2. 理论框架
### 2.1 亚里士多德修辞三要素
1. Ethos（可信度）：组织身份、责任承担、整改行动、专业资质，构建公众信任
2. Pathos（情感共情）：承认公众负面情绪、换位思考、价值共鸣，消解对立情绪
3. Logos（事实逻辑）：客观可查证事实、时间线、数据、整改方案，避免空泛辩解

### 2.2 肯尼斯·伯克新修辞认同理论
公关声明核心目标是建立组织与公众价值认同，三类认同策略：
1. 同情认同：共情公众损失与不满（危机首选）
2. 对立认同：承认矛盾、不回避问题，缩小对立距离
3. 象征认同：输出共同价值观（安全、诚信、责任）

## 3. 完整工作流程
1. 信息收集阶段：确认主体（企业/明星/政府）、事件经过、受众、核心争议、法律红线
2. 策略判断：匹配修辞三要素+认同策略，确定沟通基调（诚恳/严谨/安抚）
3. 正文生成：选用对应模板输出初稿
4. 风险复核：检查情绪缺失、事实模糊、推卸责任、法律敏感表述
5. 优化输出：调整话术，补充共情段落、可落地整改措施

## 4. 输出模板库
### 模板1：企业危机道歉声明
# XX品牌关于XX安全问题的致歉声明
各位消费者、媒体朋友：
（1）事实陈述：客观还原事件，不回避问题（Logos）
（2）共情致歉：承认公众担忧与损失（Pathos）
（3）责任承担：主动担责，不推诿（Ethos）
（4）整改措施：可落地、可监督的长期优化方案
（5）沟通渠道：公示客服、监督渠道，持续同步进展
落款：XX品牌 日期

### 模板2：明星/主播微博回应+媒体Q&A
微博短回应（简洁共情）+ 深度媒体问答（应对尖锐提问）

### 模板3：政府阶段性事故发布会口径
事实有限前提下，优先回应公众关切，区分已核实信息/待调查内容，稳定舆论

## 5. 质量评价标准（判断公关文案是否合格）
### 合格标准
1. Ethos达标：明确承担责任，给出具体整改动作，无甩锅话术
2. Pathos达标：开篇共情公众情绪，不冷漠说教
3. Logos达标：事实清晰，无模糊、虚假表述
4. 认同策略：至少使用1种共情认同，拉近公众距离
### 不合格典型问题
1. 通篇解释客观原因，未道歉、无视公众情绪（Pathos失效，如李佳琦眉笔事件）
2. 回避核心事实，话术空洞无整改方案
3. 过度官方冰冷，缺少人文共情（如早期部分事故通报）

## 6. 测试提示词（3组作业要求标准示例）
### 测试提示词1
帮我写一份餐饮品牌后厨卫生问题的道歉声明，要求真诚但规避法律风险。
> 预期输出：完整企业致歉声明，包含共情段落、全面整改、监督机制，规避赔偿、自认重大过错等敏感法律表述。

### 测试提示词2
我是明星经纪人，艺人直播发表不当言论引发全网争议，请生成微博回应全文+媒体尖锐提问Q&A清单。
> 预期输出：简短温柔微博文案+10组媒体高频问题配套稳妥回应口径。

### 测试提示词3
为市场监管局撰写安全生产事故新闻发布会口径，事故调查尚未完成，需要优先安抚群众、回应舆论质疑。
> 预期输出：政府官方严谨通报，区分已确认事实与待核查内容，兼顾安抚与严谨。

## 7. 参考案例库简述

3. 7·23动车事故王勇平发布会（政府口径负面案例：情绪共情缺失，舆论失控）
