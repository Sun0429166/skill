# 公关修辞理论框架详解
## 1 亚里士多德修辞三要素
1. Ethos（可信度）
主体身份、责任担当、官方资质、落地整改、主动接受监督，构建公众信任基础。
适用场景：企业危机道歉、政府发布会、艺人官方声明。
2. Pathos（情感共情）
承认公众负面情绪、共情损失、表达愧疚、贴合大众共同情感，消除对立。
失败典型：只解释原因、毫无道歉、无视群众痛苦。
3. Logos（事实逻辑）
清晰时间线、分条整改措施、可查证事实、闭环处置流程，拒绝空洞话术。

## 2 肯尼斯·伯克认同理论
1. 同情认同：共情大众共同担忧（食品安全、人身安全、爱国情怀）
2. 对立认同：树立共同负面对立面（违规企业、辱华品牌、不良网络内容）
3. 象征认同：输出公共正向价值观（生命至上、守法、诚信）

## 3 三类主体修辞使用差异
1. 企业：侧重Ethos（担责+整改）+同情认同
2. 明星/艺人：侧重Pathos共情+立场对立认同
3. 政府：侧重Ethos官方权威+同情认同安抚群众
